Web Development
===============
-Developing Websites


1. FrontEnd : User Interface ( Look & Feel )
         HTML : Content ( button, div, paragraph, image, table, list, form , audio )
         CSS : Styling ( Border , background, color, margin, padding )
         Bootstrap : CSS Library ( pre-defined CSS classes )
         Javascript : Behaviour of a webpage 
         Angular/React/Vue : Javascript frameworks ( RAD , CLI , Databinding , Routing , Form )

2. BackEnd : Develop REST APIs
  

3. DataBase : Holds the data ( oracle , mySql , mongoDB )
