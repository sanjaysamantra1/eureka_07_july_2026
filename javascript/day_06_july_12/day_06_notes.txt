this keyword
=============
-'this' refers to the object that is executing the current function.
-It's value is determined at runtime.


Arrow function limitations
=========================
1. arrow functions shouldn't be used as object methods.
2. arrow function shouldn't be used as event listeners


Regular function : this - how/where the function is invoked 
Arrow function : this - where the function is written (Lexical Scope)



Arrow function benefits
========================
-it uses 'this' value from Lexical scope / outer scope
